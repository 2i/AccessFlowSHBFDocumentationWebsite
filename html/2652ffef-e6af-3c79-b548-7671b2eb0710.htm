AccessFlow Namespace
This namespace contains interfaces and classes for the AccessFlow framework. The goal of AccessFlow is to parallelize operations specified in sequential form.

Base class to facilitate the implementation of IAccessContextualTScope, TThis, TChildContextual.
AwaiterForTask

Auxiliary extension methods for IChildContextualCreatorTScope, TChild.
ConstantAwaiter
Provides in implementation for an IAwaiterT whose value is already known at the time of creation of the IAwaiterT.
ConstantFuture
Provides an implementation for a IFutureT whose value is already known at the time of creation.
Future

Resolver

TaskCollector

IAccessContextTScope

IReactiveT
The return value of an asynchronous operation, which produces a result. The result can be available before the operation has completed.
IResolverT
Allows to resolve a value. This class is the counterpart to IFutureT.
ITaskCollector
