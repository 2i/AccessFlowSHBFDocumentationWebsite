Returns a Task which completes as soon as the whole RequiredAcessScope is owned by the current AccessContext so that subsequent operations may actually modify the resource managed by AccessFlow.

Remarks
If this method is used the current IAccessContextTScope must not have any child contexts, neither before nor after the invocation of this method. The method should only be invoked sequentially (usually via await)from the thread “owning” the IAccessContextTScope.
